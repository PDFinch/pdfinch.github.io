PDFinch.com Documentation

This is the documentation for the PDFinch.com site, where you can generate PDF files using our API at https://api.pdfinch.com.

Prerequisites

To use our API, you need an account with enough credit on it and a set of API keys, see your Dashboard.

Orders

You pay in advance, a minimum amount of € 5, then generate PDF files for € 0.0075 (3/4th of a cent) per page.

API

Create a set of API keys under Developers -> API Keys, then you can perform HTTP calls to our API, or use the .NET/C# Client.

See the API documentation for details.
